# Plan de Réorganisation des Dispositions Zoho CRM

Pour vous faire gagner du temps et réduire la charge mentale lors de la saisie, il est essentiel de regrouper les champs par blocs (Sections) thématiques. Voici le plan d'action en 3 phases pour vos modules principaux.

## Phase 1 : Module PROSPECTS
**Objectif :** Qualifier rapidement les leads selon leur profil (Acheteur/Vendeur) dès le premier contact.

### Sections suggérées :
1. **Informations Générales :** Prénom, Nom, Téléphone, Email, Origine du Prospect.
2. **Profil Immobilier :** Le client veut (Acheter/Vendre), Prix recherché, Type de propriété (Maison, Condo...), Propriétaire ou locataire.
3. **Question Acheteur :** Regrouper ici tous les champs dédiés aux questions pour les acheteurs.
4. **Question Vendeur :** Regrouper ici tous les champs dédiés aux questions pour les vendeurs.
5. **Détails Sociaux/Personnels :** Nom du Conjoint, Profession, Langue maternelle.
6. **Suivi & Visite :** Note de la visite, Feedback Visite, Disposition.
7. **Adresse :** Rue, Ville, Code Postal, Pays.

---

## Phase 2 : Module CONTACTS
**Objectif :** Garder une relation saine à long terme et regrouper les champs nécessaires aux automatisations d'e-mails.

### Sections suggérées :
1. **Informations de Contact :** Prénom, Nom, Téléphone, Email, Type de contact (Prospect froid, Ancien client, Partenaire).
2. **Question Acheteur :** Conserver les réponses du profil acheteur.
3. **Question Vendeur :** Conserver les réponses du profil vendeur.
4. **Marketing & Automatisations :** Courriel Bonne fête, Courriel Transaction, Date de naissance, Anniversaire de transaction.
5. **Historique Immobilier (Rappel) :** Type de propriété recherchée/vendue, Propriétaire ou locataire.
6. **Informations Secondaires :** Nom du Conjoint, Réseaux sociaux, Profession.
7. **Adresse de Correspondance :** Adresse complète.

---

## Phase 3 : Module AFFAIRES (Deals)
**Objectif :** Suivre la transaction de bout en bout avec un focus sur les délais de conditions (financement, inspection) et les intervenants.

### Sections suggérées :
1. **Détails de l'Affaire :** Nom de l'Affaire, Phase/Stade, Catégorie de Propriété, Type de Propriété, Date de clôture prévue.
2. **Finances & Commissions :** Regrouper tous les montants immobiliers, Prix vendu, Prix affiché, Formules de calculs de commission, Rétributions %.
3. **Dates Importantes (Conditions) :** Date de Financement, Date d'échéance hypo, Date Inspection, Date autres conditions.
4. **Intervenants de la transaction :** Nom Courtier Immobilier, Nom Courtier Hypothécaire, Nom Inspecteur, Nom Notaire.
5. **Notes et Particularités :** Autres conditions, Raison de la perte (si applicable).
